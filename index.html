Project Requirments - Development Details

Author: Dylan Burkey Date: 02/06/2019

Project Outline

  • Configure current DNS to use Shopify.
  • Configure development and production enviorments.
  • Content Migration
  • "Crawl"current website for a complete list of URLs.
  • 301 redirects required for all current URLs.
  • Prevent broken links
  • Prevent Google, Bing and other search engines from penalizizin/blacklisting website.
  • Content Migration.
  • Data Normalization.
  • Custom Email Functionallity.
  • Front-end data integration and web development.

DNS (Domain Name Server) Management/Migration

Current DNS records will need to be updated to load Shopify correctly.

CNAME records will need to be added for development and production enviorments.

Content Migration

Current website content and assets must be exported to integrate with Shopfiy. Access to the current CMS or Datbase will be required to export required data.

As the current website handles PIO (Personal Identifiable Information), and handles user state (Login/Logout and Register) data will need to be sanatized during the export process.

Stipulation: If the client does not have access to their Database or is unable to provide access to their CMS: Custom programming will be required to develop a "Web Scraper" to aggrigate current website content and assets.

Data Normalization

Data that is exported from the current website must be normalized and structured.

Shopfiy interacts with data using custom Database/JSON naming conventions and structures. This process requires mapping current content to Shopify's data requiremnts.

Custom Email Functionallity

Requirments:

  • Ability to push and pull data from current CRM.
  • Custom development necessary for integration between CRM and Shopify.
  • SMTP/MX record integration and custom development.

Custom development

  • Development required for CRM integration.
  • Configure client's MX and SMTP records (necessary to prevent outgoing emails flagged as spam).
  • Programming and front-end integration of custom email solution.
